# VienneyExtractor

Pour exécuter :

- (optionnel) Créer un venv puis l'activer : `python -m venv .venv` puis `source .venv/bin/activate` (pour linux)
- Installer les packages : `pip install -r requirements.txt`
- Changer les chemins des fichiers ChapitreX.pdf sources et de sortie. ⚠️ Si ils sont pareils et qu'il n'y a pas de suffixe/préfixe le poly source sera supprimé
- Exécuter le script
