# VienneyExtractor

Pour exécuter le script :

- Installer pymupdf
- (optionnel) Créer un venv puis l'activer : `python -m venv .venv` puis `source .venv/bin/activate` (pour linux)
- Installer les packages : `pip install -r requirements.txt`
- Changer les chemins des fichiers sources et de sortie. ⚠️ Si ils sont pareils et que le suffixe/préfixe est vide, le poly source sera supprimé
- Exécuter le script

Les fichiers sources doivent être nommés ChapitreXX.pdf exactement.
