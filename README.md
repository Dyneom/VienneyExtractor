# VienneyExtractor

Pour exécuter :

- (optionnel) créer un venv puis l'activer : `python -m venv .venv` puis `source .venv/bin/activate` (pour linux)
- installer les packages : `pip install -r requirements.txt`
- Changer les chemins des fichiers ChapitreX.pdf sources et de sortie. ⚠️ Si ils sont pareils le poly source sera supprimé
- Exécuter le script
